# Contributing

The main goal of this project is to build trustworthy subtitle sanitation and diagnostics tooling.

The most valuable contributions right now are:

* malformed subtitle files
* broken exports
* corrupted subtitle examples
* workflow pain reports
* editor compatibility issues

If you encounter broken subtitle files:

1. Open a GitHub issue
2. Describe the failure
3. Attach the subtitle file if possible
4. Mention the software/export source

Current focus:

* diagnostics
* corruption detection
* safe repair workflows
* subtitle linting
